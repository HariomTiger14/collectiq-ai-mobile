# Product Language gap analysis

Available and approved in Flutter at `c6bf080`: `PackLoxHeader` 1.0.1, `PackLoxHero` 1.0.1, `PackLoxEntryTile` 1.0.0, `PackLoxButton` 1.0.0 plus tokens; Scanner S01 is the validated usage reference.

Gaps before broad reconstruction: bootstrap/splash composition; onboarding pager/education/permission pattern; auth fields, validation and status banners; shell/navigation and system-bar contract; generic loading/empty/error/offline/permission/fatal-state families; cards/lists/data displays; dialogs/sheets; camera controls and capture guidance beyond S01; analysis progress/result families; portfolio grid/list/filter; detail gallery/metrics/actions; search; notification items; settings rows/toggles/account identity.

Do not infer these families from the Design Bible alone. Each needs Studio composition, responsive/accessibility specification, implementation package, runtime evidence and approval before reuse.
