# Surface And Colour Comparison

Authority: approved S02 Empty Collection crop and Home master board.
Runtime: fresh 01_launch.png.

Shared traits:

- Both are dark surfaces.
- Both use blue for primary action emphasis.
- Both use rounded card surfaces and pale text on dark backgrounds.

Differences:

- Approved S02 uses a near-black root with a single dark empty-state card and a visible illustration; runtime uses a bright blue gradient Hero as the dominant first-viewport surface.
- Approved S02 primary button sits inside the empty-state card; runtime primary button sits inside the Hero, with two large action cards below it.
- Approved S02 category tiles use compact dark chips/cards with icon accents; runtime has no category tiles in the empty first viewport.
- Approved S02 bottom nav is compact and includes Search; runtime bottom nav is taller, glass-like, and excludes Search.
- Approved board supports status colors for good, amber, warning, and unavailable states; runtime empty state exposes only blue emphasis and muted surfaces.

Conclusion:

The runtime follows PLX-PL-1.0 tokens for background, surface, raised surface, border, text, blue, cyan, and hero gradient, but it is not a direct S02 Home board implementation.
