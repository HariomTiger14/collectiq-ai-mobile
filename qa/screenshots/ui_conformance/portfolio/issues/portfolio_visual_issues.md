# Portfolio Visual Issues

## UI-AUDIT-PORTFOLIO-001

Severity: High

Issue: Portfolio lacks direct approved visual references for populated, empty, search no-results, filter sheet, and sort sheet states.

Recommended correction: Approve a Portfolio state matrix before freeze.

## UI-AUDIT-PORTFOLIO-002

Severity: Medium

Issue: Filter and sort bottom sheets appear as screen-local runtime states rather than approved shared-state primitives.

Recommended correction: Promote sheet styling into shared Product Language or document Portfolio-specific exceptions.
